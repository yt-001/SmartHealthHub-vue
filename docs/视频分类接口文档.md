# 视频分类接口文档

## 1. 分页查询视频分类

- **路径**: `POST /video-categories/page`
- **请求体**: 
```json
{
  "pageNum": 1,
  "pageSize": 10,
  "query": {
    "name": "分类名称",
    "isEnabled": 1
  }
}
```
- **返回体**: 
```json
{
  "code": 200,
  "message": "success",
  "data": {
    "pageNum": 1,
    "pageSize": 10,
    "total": 100,
    "pages": 10,
    "list": [
      {
        "id": 1,
        "name": "健身教学",
        "description": "各类健身动作教学视频",
        "sortOrder": 1,
        "isEnabled": 1
      }
    ]
  }
}
```

## 2. 获取所有启用的视频分类（简化信息）

- **路径**: `GET /video-categories/simple-list`
- **请求体**: 无
- **返回体**: 
```json
{
  "code": 200,
  "message": "success",
  "data": [
    {
      "id": 1,
      "name": "健身教学"
    }
  ]
}
```

{

## 4. 根据ID获取视频分类详情

- **路径**: `GET /video-categories/{id}`
- **请求体**: 无
- **返回体**: 
```json
{
  "code": 200,
  "message": "success",
  "data": {
    "id": 1,
    "name": "健身教学",
    "description": "各类健身动作教学视频",
    "sortOrder": 1,
    "isEnabled": 1
  }
}
```

## 5. 创建视频分类

- **路径**: `POST /video-categories/create`
- **请求体**: 
```json
{
  "name": "新分类名称",
  "description": "分类描述",
  "sortOrder": 1,
  "isEnabled": 1
}
```
- **返回体**: 
```json
{
  "code": 200,
  "message": "分类创建成功",
  "data": null
}
```

## 6. 更新视频分类

- **路径**: `PUT /video-categories/update`
- **请求体**: 
```json
{
  "id": 1,
  "name": "更新后的分类名称",
  "description": "更新后的分类描述",
  "sortOrder": 2,
  "isEnabled": 1
}
```
- **返回体**: 
```json
{
  "code": 200,
  "message": "分类更新成功",
  "data": null
}
```

## 7. 删除视频分类

- **路径**: `DELETE /video-categories/delete/{id}`
- **请求体**: 无
- **返回体**: 
```json
{
  "code": 200,
  "message": "分类删除成功",
  "data": null
}
```
