# 文章分类接口文档

## 1. 分页查询文章分类

- **路径**: `POST /article-categories/page`
- **请求体**: 
```json
{
  "pageNum": 1,
  "pageSize": 10,
  "query": {
    "name": "分类名称",
    "isEnabled": 1
  }
}
```
- **返回体**: 
```json
{
  "code": 200,
  "message": "success",
  "data": {
    "pageNum": 1,
    "pageSize": 10,
    "total": 100,
    "pages": 10,
    "list": [
      {
        "id": 1,
        "name": "健康饮食",
        "description": "关于营养搭配、健康饮食习惯等方面的文章",
        "sortOrder": 1,
        "isEnabled": 1
      }
    ]
  }
}
```

## 2. 获取所有启用的文章分类（简化信息）

- **路径**: `GET /article-categories/simple-list`
- **请求体**: 无
- **返回体**: 
```json
{
  "code": 200,
  "message": "success",
  "data": [
    {
      "id": 1,
      "name": "健康饮食"
    }
  ]
}
```

## 4. 根据ID获取文章分类详情

- **路径**: `GET /article-categories/{id}`
- **请求体**: 无
- **返回体**: 
```json
{
  "code": 200,
  "message": "success",
  "data": {
    "id": 1,
    "name": "健康饮食",
    "description": "关于营养搭配、健康饮食习惯等方面的文章",
    "sortOrder": 1,
    "isEnabled": 1
  }
}
```

## 5. 创建文章分类

- **路径**: `POST /article-categories/create`
- **请求体**: 
```json
{
  "name": "新分类名称",
  "description": "分类描述",
  "sortOrder": 1,
  "isEnabled": 1
}
```
- **返回体**: 
```json
{
  "code": 200,
  "message": "分类创建成功",
  "data": null
}
```

## 6. 更新文章分类

- **路径**: `PUT /article-categories/update`
- **请求体**: 
```json
{
  "id": 1,
  "name": "更新后的分类名称",
  "description": "更新后的分类描述",
  "sortOrder": 2,
  "isEnabled": 1
}
```
- **返回体**: 
```json
{
  "code": 200,
  "message": "分类更新成功",
  "data": null
}
```

## 7. 删除文章分类

- **路径**: `DELETE /article-categories/delete/{id}`
- **请求体**: 无
- **返回体**: 
```json
{
  "code": 200,
  "message": "分类删除成功",
  "data": null
}
```
